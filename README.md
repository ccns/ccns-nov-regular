# CCNS NOV REGULAR

[Sample Contract](https://goerli.etherscan.io/address/0xf377cf053ed2ccadeb144962e82db468cdc2cf86)

## Development

1. clone

```bash
git clone https://github.com/wiasliaw/ccns-nov-regular.git
```

2. install

```bash
cd ccns-nov-regular
npm install
```

3. compile contract

```bash
npm run compile
```

4. test contract

```bash
npm run test
```

5. deploy and mint

```bash
npm run deploy:goerli
npm run mint:goerli
```
